## How to run project

```sh
git clone git@github.com:Big-Sh4rk/RN-Slack-Clone.git
cd slack-clone-react-native

# Install npm dependencies
yarn

# Link the assets

npx react-native link

# Install pod dependencies
npx pod-install

# Run on ios simulator
npx react-native run-ios
```
